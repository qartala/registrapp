Aplicacion de asistencia.

Aplicación dedicada para los alumnos de Duoc UC ,Esta aplicacion cuenta con:

Login

Vista compuesta por fomulario para ser completado con los datos de acceso del docente o alumno.

Restablecer Contraseña

Vista habilitada para completar el formulario con contraseña nueva.

Inicio Alumno

Vista "Inicio" de la aplicacion como Alumno,esta vista contiene una carta con informacion correspondiente a la aplicacion, Ademas tiene habilitada la opcion "Scanear" lo cual permite el escaneo de un codigo Qr.

Scanear

Vista de camara habilitada para escanear codigo Qr.

Menu

Apartado habilitado para cerrar sesion, Junto al nombre del Usuario ingresado

Escanear Qr para descargar Aplicacion.
